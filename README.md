# 📘 정보처리산업기사 실기 암기용 요약

> 시험 직전 한눈에 보는 핵심 요약본 (V-모델, 테스트, 운영체제, 네트워크, 보안, DB, 자료구조, 프로그래밍)

---

## 1️⃣ 소프트웨어 개발 생명주기 (V-모델)

| 단계 | 설명 |
|------|------|
| 사용자 요구사항 | 최종 사용자가 필요로 하는 기능 정의 |
| 분석 | 요구사항 분석, 기능 분해, 모델링 |
| 설계 | 시스템 구조 설계 (상위/하위 설계) |
| 구현 | 소스코드 작성, 모듈 개발 |
| 단위 테스트 | 모듈 단위 검증 <br>**블랙박스**: 사용자 요구 기반 → 경계값 분석, 동치분할, 원인-효과 그래프 <br>**화이트박스**: 내부 구조 논리 → 구조, 반복, 조건 |
| 통합 테스트 | 모듈 결합 검증 <br>점진적 방식(상향식/하향식) vs 비점진적(빅뱅) |
| 시스템 테스트 | 전체 시스템 기능 + 성능 + 장애 처리 검증 |
| 인수 테스트 | 최종 사용자 승인 <br>**알파**: 개발자 앞 <br>**베타**: 실제 사용자 환경 |

---

## 2️⃣ 테스트 기법

- **블랙박스 테스트**
  - 사용자 요구사항 명세서 기반
  - 기법: 경계값 분석, 동치분할, 원인-효과 그래프
- **화이트박스 테스트**
  - 내부 구조 논리 기반
  - 기법: 구조, 반복, 조건

## 3️⃣ 자료구조

| 구조 | 특징 |
|------|------|
| 스택 (LIFO) | 후입선출 |
| 큐 (FIFO) | 선입선출 |
| 덱 (Deque) | 양쪽 삽입/삭제 가능 |
| 링크드리스트 | 포인터로 연결된 노드 집합 |
| 해시테이블 | 키 → 해시 → 값 |
| 트리 | 계층적 구조, 탐색 용이 |
| 그래프 | 정점과 간선, BFS/DFS 가능 |

## 4️⃣ 프로그래밍 & 디자인 패턴

### 디자인 패턴 핵심

| 패턴 | 유형 | 설명 |
|------|------|------|
| 싱글톤 (Singleton) | 생성 | 인스턴스를 하나만 생성, 전역 접근 가능 |
| 팩토리 (Factory) | 생성 | 객체 생성 코드 캡슐화, 필요에 따라 다른 객체 생성 |
| 옵저버 (Observer) | 행동 | 상태 변화 시 관련 객체에 자동 알림 |
| 전략 (Strategy) | 행동 | 알고리즘을 캡슐화, 필요 시 교체 가능 |
| 데코레이터 (Decorator) | 구조 | 객체 기능 동적으로 추가/확장 |
| 프록시 (Proxy) | 구조 | 접근 제어, 대리자 역할 수행 |
| 컴포지트 (Composite) | 구조 | 객체를 트리 구조로 구성, 부분-전체 동일 처리 |
| 템플릿 메서드 (Template Method) | 행동 | 알고리즘 구조 정의, 세부 단계 서브클래스에서 구현 |

## 5️⃣ 운영체제 & CPU 스케줄링

- **CPU 스케줄링**
  - FCFS: 선입선출
  - SJF: 실행시간 짧은 순서 우선
  - RR: 시분할, 고정 시간량 CPU 순환
- **운영 방식**
  - 다중 프로그래밍: 하나의 주기억장치에 여러 프로그램 동시 처리
  - 시분할: 시간 단위로 CPU 순환
  - 다중 처리: 여러 CPU 사용
  - 일괄 처리: 작업 모아서 처리
- **교착상태(데드락) 발생 조건**
  - 상호배제(Mutual Exclusion)
  - 점유 및 대기(Hold and Wait)
  - 비선점(No Preemption)
  - 순환대기(Circular Wait)

## 6️⃣ 네트워크

- **TCP/IP 계층**
  - 응용 → 전송 → 인터넷 → 네트워크 액세스
- **IP 클래스**
  - A: 1~127 / 255.0.0.0
  - B: 128~191 / 255.255.0.0
  - C: 192~223 / 255.255.255.0
- **IPv4 vs IPv6**
  - IPv4: 32비트, 유니캐스트/브로드캐스트/멀티캐스트
  - IPv6: 128비트, 유니캐스트/애니캐스트/멀티캐스트, QoS 향상
- **주요 포트번호**

| 포트번호 | 프로토콜 | 설명 |
|----------|----------|------|
| 21       | FTP      | 파일 전송 |
| 22       | SSH      | 보안 셸 접속 |
| 23       | TELNET   | 원격 접속 |
| 25       | SMTP     | 메일 송신 |
| 53       | DNS      | 도메인 이름 해석 |
| 80       | HTTP     | 웹 서비스 |
| 110      | POP3     | 메일 수신 |
| 143      | IMAP     | 메일 수신/동기화 |
| 161      | SNMP     | 네트워크 관리 |
| 162      | SNMP     | 트랩 메시지 수신 |
| 443      | HTTPS    | 보안 웹 서비스 |

- NAT: 사설 ↔ 공인 IP 변환  
- ARP/RARP: IP ↔ MAC 주소 변환  
- RIP: 최단 경로 탐색, 최대 홉수 15

## 8️⃣ 데이터베이스 & SQL

- **SQL 명령어**
  - DDL: CREATE, ALTER, DROP
  - DML: INSERT, SELECT, UPDATE, DELETE
  - DCL: GRANT, REVOKE
  - TCL: COMMIT, ROLLBACK
- **트랜잭션 ACID**
  - A: 원자성, C: 일관성, I: 독립성, D: 지속성
- **정규화**
  - 1NF: 속성 원자값
  - 2NF: 부분 함수 종속 제거
  - 3NF: 이행적 종속 제거
  - BCNF: 결정자가 후보키
  - 4NF: 다치 종속 제거
- **키 및 무결성**
  - PK, FK, 슈퍼키, 후보키
  - 개체 무결성, 참조 무결성
- **E-R 다이어그램**
  - 사각형: 개체, 마름모: 관계, 타원: 속성
- **릴레이션 연산**
  - SELECT(σ), PROJECT(π), JOIN(▷◁), UNION, INTERSECTION, DIFFERENCE
